# Industrial Predictive Analytics Pipeline

A machine learning pipeline for industrial sensor monitoring and predictive analytics.

## Features

- Data preprocessing using Pandas
- Anomaly detection using Isolation Forest
- Predictive modelling using Random Forest
- Model evaluation using RMSE
- Data visualisation using Matplotlib

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Project Objective

The aim of this project is to demonstrate an end-to-end machine learning workflow for industrial sensor data, including anomaly detection and predictive maintenance analysis.
